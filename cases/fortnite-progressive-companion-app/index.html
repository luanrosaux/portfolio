Game UX · PWA Concept

Fortnite Progressive Companion App

Exploring how a progressive companion app could extend Fortnite’s engagement beyond gameplay.

Role
UI/UX Designer
Timeline
Nov 2022 – Dec 2022
Sector
Game UX
Focus
Companion app, PWA and engagement loops

context

The concept explored a build-once companion experience with real-time updates, character customization, item shop access and player features outside the console experience.

The challenge

Companion apps can strengthen engagement, but native app development creates distribution friction. The project explored whether a PWA could deliver similar value with less platform dependency.

product visuals

Mobile and desktop screens showing how the companion app extends player actions, customization and real-time game content.

process

How the experience was shaped.

01

Competitive Analysis

Benchmarked companion apps from major games to identify useful patterns and engagement opportunities.

02

PWA Strategy

Explored progressive web app advantages such as build once, run everywhere, no download friction and platform flexibility.

03

Prototype and Testing

Designed mobile and web flows for login, locker, battle pass, item shop, quests, map and profile.

impact

What changed after the redesign.

PWA
Alternative to native companion app distribution.
Real-time
Map, store and game content access outside gameplay.
Validated
Concept refined through user testing feedback.
